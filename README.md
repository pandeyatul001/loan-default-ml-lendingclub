# Predicting Loan Default using Machine Learning (LendingClub)

This project develops and compares machine learning models to predict loan default risk using the LendingClub dataset.

## Models
- Logistic Regression
- Random Forest
- Neural Network (MLP)

## Dataset
LendingClub accepted loans (2007–2018)  
Sample: 40,000 loans

## Key Findings
- Neural Network achieved highest predictive performance
- Logistic Regression provided strongest interpretability
- Interest rate, credit sub-grade, and loan term were dominant predictors
